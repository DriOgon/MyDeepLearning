# Markdown常用语法

## 一. 标题

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 二.段落格式 

```
* 列表 （可嵌套）
*斜体*   
**粗体**  
***粗斜体***  
---分割线  
~~删除线~~    
<u>下划线</u>
==高亮==
```
## 三.代码 

```
`段落一行的代码`

```代码块``` 
```
## 四.超链接  

```
[链接名称](网址)
```
示例：[这是一个超链接,点我](https://www.bilibili.com/) 
- - -

```
![文本属性](图片地址)
```
